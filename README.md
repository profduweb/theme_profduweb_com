# Profduweb WordPress Theme

Theme WordPress minimaliste pour ProfduWeb.

## Version

4.3.2

## Points inclus

- Balises SEO dynamiques.
- Canonical URL.
- Open Graph et Twitter Card.
- Donnees structurees JSON-LD Schema.org.
- Nettoyage du paquet de production.
- Durcissement du rendu des SVG personnalises dans les menus.

## Changements recents

### 4.3.2

- Affichage mobile de la page d'accueil corrige: une seule actualite par ligne pour garder les titres lisibles.
- Badge de commentaires de la page d'accueil repositionne dans l'image de chaque carte.
- Libelle du badge de commentaires rendu plus clair avec le nombre et le mot `commentaire(s)`.
- Paquet de production nettoye pour exclure les fichiers temporaires et sauvegardes locales.

## Installation

Compresser le contenu du depot en ZIP, puis l'installer dans WordPress via **Apparence > Themes > Ajouter > Televerser un theme**.
