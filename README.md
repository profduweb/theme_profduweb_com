# Profduweb WordPress Theme

Theme WordPress minimaliste pour ProfduWeb.

## Version

4.2

## Points inclus

- Balises SEO dynamiques.
- Canonical URL.
- Open Graph et Twitter Card.
- Donnees structurees JSON-LD Schema.org.
- Nettoyage du paquet de production.
- Durcissement du rendu des SVG personnalises dans les menus.

## Installation

Compresser le contenu du depot en ZIP, puis l'installer dans WordPress via **Apparence > Themes > Ajouter > Televerser un theme**.
